#Edison

The prototype is wrapped into streamlit UI and deployed [here](https://edison.streamlit.app/)

## Installation

```bash
git clone https://github.com/xnileshtiwari/Edison.git
```
```bash
cd Edison
```
```bash
pip3 install requirements.txt
```

```bash
streamlit run Product_CO2.py
```
